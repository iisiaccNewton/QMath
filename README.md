# QMath
QMath is a Mathematica package specifically designed for **particle physics** and **quantum field theory**. 
It is capable of performing general particle physics calculations, 
such as **Dirac matrix tracing**, as well as computing **Feynman diagrams** at both the **tree level** and **one-loop** level.

# QMath's Google AI Persona
We have designed an AI assistant [QMath](https://gemini.google.com/gem/12Z1z6YGOG8wiexDUdJnXnqI350teJCLV?usp=sharing) specifically to answer beginner questions regarding the QMath package. As the assistant is powered by a Large Language Model (LLM), it is inevitably subject to AI hallucinations. Please do not rely entirely on its responses. When necessary, we encourage you to consult the source code, which has been designed with simplicity in mind—anyone with a basic understanding of Wolfram Language should find it easy to comprehend.
